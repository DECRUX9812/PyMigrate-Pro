## Reporting a Vulnerability

We take the security of PyMigrate Pro seriously. If you have discovered a security vulnerability, please report it to us privately.

**Do not open a public issue.**

Instead, please email **<decrux9812@users.noreply.github.com>** with a detailed description of the issue. We will review it and respond as soon as possible.
