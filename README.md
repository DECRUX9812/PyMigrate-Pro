# PyMigrate Pro

![CI](https://github.com/DECRUX9812/PyMigrate-Pro/actions/workflows/ci.yml/badge.svg)

PyMigrate Pro is a lightweight, reliable migration tool for User Profiles on Windows. It is designed to be a safer, portfolio-ready alternative to tools like Transwiz.

## ‚ú® Features

- **Batch Processing**: Migrate multiple users at once.
- **Smart Registry Export**: Captures Outlook Profiles and Desktop Wallpaper safely.
- **Mail Discovery**: Automatically finds `.pst/.ost` files and Thunderbird profiles.
- **Portable**: Can be built into a single `.exe` file.
